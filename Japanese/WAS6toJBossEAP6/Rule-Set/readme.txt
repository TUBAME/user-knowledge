ナレッジ管理ツールよりWAS6toEAP6.xmlを利用(編集）する場合は以下の手順にしたがい、
TUBAMEのインストール及びwas-programlistingのコピーを行ってください
 
  * TUBAMEインストール
   * http://tubame.github.io/migration-tool/index_ja.html
     
  * programlistingのコピー
   * was-programlistingフォルダは、TUBAMEのナレッジ管理ツールが配置されている
     フォルダ(...eclipse/plugins/tubame.knowhow-*/resources/xsl)にコピーしてください。
     
  * ナレッジベース検索ツールを利用する場合は、以下の手順に従い、Guidelineフォルダ
    にあるコンテンツをコピーしてください。
    * ...eclipse/plugins/tubame.portability-*/resources/guideフォルダを作成後、
     Guidelineフォルダにあるindex.html及びcssフォルダ、imagesフォルダをコピーしてください
